# Screenshots untuk Portfolio

Tambahkan screenshot di folder ini, lalu embed di README utama.

## Cara ambil screenshot

1. Jalankan `npm run web`
2. Login ke app
3. Buka tiap layar di bawah
4. Screenshot (Windows: `Win + Shift + S`)

## Daftar yang disarankan

| File | Isi layar |
|------|-----------|
| `dashboard.png` | Beranda — saldo & transaksi terbaru |
| `transactions.png` | Daftar transaksi + filter kategori |
| `budget.png` | Anggaran dengan progress bar |
| `reports.png` | Laporan income vs expense |
| `settings.png` | Pengaturan — tema, bahasa, mata uang |
| `pin-lock.png` | Layar kunci PIN (kotak digit) |
| `recurring.png` | Daftar transaksi berulang |
